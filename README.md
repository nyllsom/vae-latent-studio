# vae-latent-studio
基于 TensorFlow.js 的浏览器端 VAE latent space 小实验

## 文件
- `index.html`：入口页
- `vae_frontend.html`：VAE 版本
- `ae_frontend.html`：AE 版本

## 功能
- 浏览器内训练
- 图像重建
- 随机采样
- latent 散点图
- latent 直方图
- latent sweep
- manifold exploration

## 使用
直接双击 `index.html`，用浏览器打开即可。

注意：
- 需要联网
- 页面会从 CDN 加载依赖
- 页面会在线读取 MNIST 数据
